# AgroMindAI 🌱

AgroMindAI is an AI-powered agriculture assistant designed to help farmers make informed decisions through crop recommendations, weather insights, and agricultural guidance.

## Features
- Crop recommendation system
- Weather-based farming insights
- AI-powered agricultural assistance
- User-friendly web interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- AI Agents

## Project Structure

```
AgroMindAI/
├── agents/
├── data/
├── static/
├── templates/
├── app.py
└── requirements.txt
```

## Installation

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```
